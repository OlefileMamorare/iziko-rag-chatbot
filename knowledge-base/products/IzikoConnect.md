# IzikoConnect

## Summary

IzikoConnect is Iziko Cloud Solutions' unified cloud collaboration platform. It bundles enterprise-grade messaging, video conferencing, file sharing, and shared workspaces into a single POPIA-compliant offering hosted in South African data centres.

## Target Customers

Mid-sized to large South African enterprises (200-5,000 employees) that need a unified collaboration suite with local data residency, low-latency performance over local fibre, and pricing in Rand. Particularly strong fit for financial services, retail, and provincial government.

## Key Features

- HD video meetings for up to 500 participants
- Persistent team channels with message-level encryption
- File sharing with version history (retention configurable: 30, 90, or 365 days)
- SAML 2.0 single sign-on, including Microsoft Entra ID and Okta
- Native mobile apps for iOS and Android, optimised for low-bandwidth conditions
- 99.9% monthly uptime SLA, measured against the production service status page

## Pricing Tiers (per month, ex VAT, billed in ZAR)

- **Starter Tier** - R 18,000 per month. Up to 100 users, 8x5 support, 30-day file retention.
- **Standard Tier** - R 45,000 per month. Up to 500 users, 24x5 support, 90-day file retention.
- **Enterprise Tier** - R 95,000 per month. Unlimited users, 24x7 support, 365-day file retention, dedicated customer success manager.
